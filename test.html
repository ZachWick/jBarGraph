Use

Create a bar graph using the default options: 
      jQuery("#graph_container_element").bar_graph({
      data:[1,2,3,4],
      labels:["A","B","C","D"]
      });
Create a bar graph using some non-default options: 
      jQuery("#graph_container_element").bar_graph({
      style:"vertical",
      vAxis:true,
      data:[1,2,3,4],
      labels:["A","B","C","D"],
      correct:["C"],
      colorByCorrect:true
      });

Options

Option NameDefault ValueOther Accepted ValuesNotes
data[ ]N/AAn array of the raw data values to graph
labels[ ]N/AAn array of strings to use as labels
labelStyle"split""text","value","full"'text' => Just the label text is displayed.
'value' => Just the data value is displayed.
"full" => both text and value are displayed as "text (value)"
labelDisplay"scale""hover","static""hover" => label shown as alt text/title attribute of each bar.
"scale" => label shown on bar, and font-size scales as barwidth / 4.
labelPos"inside""outside""inside" => labels are shown inside of the colored bars.
"outside" => labels are shown outside of the colored bars.
colorByCorrectfalsetrueIf true, bars corresponding to incorrect options are blue and bars corresponding to correct options are lime
correct[ ] N/AAn array of the labels of the bars that are correct options
barStyle"fancy""plain""fancy" => rounded corners on bars.
"plain" => square corners on bars.
style"horizontal""vertical""horizontal" => bar graph where the bars are horizontal.
"vertical" => bar graph where the bars are vertical.
vAxisfalsetrueIf true, shows a scale on the vertical axis. Only applicable if style == 'vertical'
vAxisStepDivisor5
vAxisSteps10The number of steps to display from 0 to max(data)
